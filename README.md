# アプリケーションを作成した背景

私が所属している野球チームのホームページが無かったのでプログラミングを学んだら作成しようと  
考えておりました。また作成に伴い様々なチームが試合を申し込みたいと思えるようなホームページを  
作りたいと思い作成しました。
